## Commando's

* `mkdocs new [dir-name]` - Maak een nieuw project aan.
* `mkdocs serve` - Start de live-herlaad docs server.
* `mkdocs build` - Bouw de documentatiesite.
* `mkdocs -h` - Help-bericht afdrukken en afsluiten.
